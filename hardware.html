Hardware

Introduction

Welcome to the Hardware page! This page is here to give a little information about computer hardware. Here you can learn about the motherboard, CPU, RAM, and much more!

Motherboard

The motherboard is a printed circuit board that contains the main components of a computer. Some components that the motherboard holds are the CPU, RAM, USB slot, etc. The motherboard is powered by the power supply unit (PSU). The purpose of the motherboard is to connect and manage all the computer's subsystems so they can communicate with each other.

CPU

The Central Processing Unit or the CPU is commonly referred to as the brain of the computer. The CPU handles all the instructions received from hardware or software running on the computer.

RAM

The Random Access Memory, or RAM is a type of computer memory that can be accessed randomly. It is considered "random access" because you can access any memory cell.

Hard Drive

The hard drive is where the software, operating system, and much more get stored. Without a hard drive a computer would not be able to function.

Keyboard

The keyboard is what people use to to input text into a copmuter. A keyboard contains symbols, letters, and numbers. There are also gaming computers that are bulit specifically for gamers. These mouses generally are more durable, have better painted letters, and custom/extra keys.

Mouse

The mouse is the what people use to move the cursor around the screen. With the mouse you can select things, and drag objects. As well as keyboards, there are gaming mouses and normal mouses. Most gaming mouses contain a customizable DPI button, and custom keys on the side of the mouse. Not all gaming mouses or mouses are the same, some people like bigger mouses with custom buttons on the side and a wired cord, and some like smaller mouses that are wireless.

Monitor

The monitor is probably the most important thing to have while using a computer. Without the monitor you can not see what is going on! The monitor is connected to the motherboard, and allows you to see what we are doing on the computer. People can buy monitors that are different sizes, like a 20 inch monitor or 24 inch monitor.

How do all these parts communicate with each other?

All of these components are essential to having a computer, and they all work as a team to make the computer function. All of these parts work by being connected to each other. The motherboard connects all of the components together. The hard drive is where all the information for the computer is stored, and RAM is the memory card. The monitor is then connected to the computer to make all the content show up for the user to see. The user can then use the mouse and the keyboard to play around, and use the computer.
