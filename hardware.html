Hardware

Introduction

Welcome to the Hardware page! This page is here to give a little information about computer hardware. Here you can learn about the motherboard, CPU, RAM, and much more!

Motherboard

The motherboard is a printed circuit board that contains the main components of a computer. Some components that the motherboard holds are the CPU, RAM, memory, etc. The purpose of the motherboard is to connect and manage all the computer's subsystems so they can communicate with each other.

CPU

The Central Processing Unit or the CPU is commonly referred to as the brain of the computer. The CPU handles all the instructions recieved from hardware or software running on the computer.

RAM

The Random Access Memory, or RAM is a type of computer memory that can be accessed randomly. It is considered "random access" because you can access any memory cell.

Hard Drive

The hard drive is the computer's storage. It pretty much stores all the information on your computer, from pictures to your preferences.

Keyboard

The keyboard is what people use to to insert text into a copmuter. A keyboard contains symbols, letters, and numbers.

Mouse

The mouse is the what people use to move the cursor around the screen. With the mouse you can select things, and drag things.
